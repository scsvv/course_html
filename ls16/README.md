на место условия мы используем логические операции: 
и, или, нет, равно, не равно, больше, больше либо равно, меньше, меньше либо равно. 

null - то что присваивает программист.  
undefined - то что ставит программа (интепритатор)
0, null, undefined == false

&& - and - два тру
|| - or - один тру
! - not
